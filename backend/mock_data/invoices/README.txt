Place mock invoice files in this folder for auto-ingestion at Start Agent.

Supported formats: .pdf, .png, .jpg, .jpeg
Recommended naming to map correctly:
- TixiTaxi_Invoice.pdf (or filename containing: tixi / taxi / transport)
- MealInvoice.pdf (or filename containing: meal)

When present, the backend will auto-feed these files to the agent in mock mode.
